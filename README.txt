Tarea 2 BASE DE DATOS
=============================================
--------------------------------------------------------------------
IMPORTANTE SOBRE PAQUETES USADOS
--------------------------------------------------------------------
- Para el desarrollo de esta tarea se utilizó el framework XAMPP en
su versión 8.2.12 y su control de panel versión 3.3.0, se recomienda
encarecidamente utilizar este framework en esta versión.

--------------------------------------------------------------------
AMBIENTE DE EJECUCION USADO
--------------------------------------------------------------------
- Sistema operativo: Windows 11
- Ejecución del código: 
  + Visual Studio Code (Para la escritura de los
    archivos .php y sus derivados).
  + Apache 2.4.58
  + MariaDB 10.4.32
  + PHP 8.2.12 (VS16 X86 64bit thread safe) + PEAR
  + phpMyAdmin 5.2.1

--------------------------------------------------------------------
EJECUCIÓN
--------------------------------------------------------------------
- Para llevar a cabo una correcta ejecución se requiere posicionar
  las carpetas "css", "img" y "php" dentro de htdocs creando una
  carpeta con nombre tarea2.(ejemplo de ruta: C:\xampp\htdocs\tarea2)

--------------------------------------------------------------------
CONSIDERACIONES AL MOMENTO DE EJECUTAR
--------------------------------------------------------------------
- Para que todo funcione de manera correcta se deberá ingresar a 
  localhost, donde una vez dentro se deberá entrar en la sección 
  phpmyadmin, para así crear una base de datos la cual deberá ser
  nombrada como "tarea2".

- Una vez creada la base de datos esta se deberá poblar con el archivo
  tablasActualizadas.sql, esto se realiza en la sección "importar".

- Por último, para comenzar a utilizar la aplicación web se deberá
  ingresar en http://localhost/tarea2/php/sesiones.php.
